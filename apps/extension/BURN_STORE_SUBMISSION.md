## Single Purpose Description

### What is the single purpose of your extension?

Burn's single purpose is to provide **anonymous AI chat access without requiring user accounts**. The extension enables users to:

1. **Chat with AI anonymously** - No login, no account, no personal data
2. **Subscribe as guests** - Anonymous Stripe checkout for premium features
3. **Create private agents** - Custom AI assistants without identity tracking
4. **Burn sessions** - Ephemeral conversations that leave no trace

All functionality serves this core purpose: providing full AI capabilities while maintaining complete user anonymity and privacy.

### How does this extension accomplish its single purpose?

**Anonymous Access:**

- No authentication required
- Browser fingerprinting for session continuity only
- Local storage for credits and preferences
- No server-side user profiles

**Guest Subscriptions:**

- Stripe Checkout with guest mode
- Anonymous payment processing
- Credits stored in browser localStorage
- No subscription tied to user identity

**Private AI Chat:**

- Direct API calls to AI providers
- No conversation history stored
- Ephemeral sessions
- All data deleted on tab close

**Burn Functionality:**

- Clear localStorage on session end
- No cookies or persistent tracking
- No server-side data retention
- Complete privacy reset

**Why needed:** Store credits, preferences, and session data locally
