3. **Test AI Chat**:
   - Type "Explain quantum computing in simple terms" in the chat
   - The AI will respond immediately using GPT-4, Claude, or Gemini
   - Try switching between different AI models using the model selector
4. **Test Context Menu** (Right-click features):
   - Go to any webpage (e.g., Wikipedia article)
   - Select any paragraph of text
   - Right-click and choose "Explain" or "Summarize"
   - The sidebar opens with AI-generated results
